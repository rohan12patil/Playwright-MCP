
## Available MCP Tools

### Navigation
- `browser_navigate` - Navigate to a URL
- `browser_navigate_back` - Go to previous page
- `browser_navigate_forward` - Go to next page

### Interaction
- `browser_click` - Click an element
- `browser_type` - Type text into input fields
- `browser_select_option` - Select dropdown options
- `browser_drag` - Drag and drop elements
- `browser_hover` - Hover over elements
- `browser_press_key` - Press keyboard keys
- `browser_file_upload` - Upload files
- `browser_handle_dialog` - Handle browser dialogs

### State & Observation
- `browser_snapshot` - Get accessibility tree snapshot (accessibility info)
- `browser_wait_for` - Wait for text or delay
- `browser_network_requests` - Monitor network activity
- `browser_console_messages` - Retrieve console logs

### Artifacts & Testing
- `browser_take_screenshot` - Capture page screenshots
- `browser_pdf_save` - Save page as PDF
- `browser_generate_playwright_test` - Generate test code

### Vision Mode (Coordinate-based)
- `browser_screen_capture` - Capture screen image
- `browser_screen_click` - Click coordinates
- `browser_screen_drag` - Drag from coordinates
- `browser_screen_type` - Type from coordinates
- `browser_screen_move_mouse` - Move mouse to coordinates

### Tab & Session Management
- `browser_tab_list` - List open tabs
- `browser_tab_new` - Open new tab
- `browser_tab_select` - Switch to tab
- `browser_tab_close` - Close tab
- `browser_resize` - Resize window
- `browser_close` - Close browser

### Utilities
- `browser_install` - Install required browser binaries
